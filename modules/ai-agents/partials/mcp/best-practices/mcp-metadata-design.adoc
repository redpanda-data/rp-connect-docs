=== Choose a clear tool name

AI clients see the `label` value when selecting tools, so make it descriptive and consistent.

The name should reflect the tool's primary action and target resource. Focus on clarity over brevity.

.Tool naming examples
[cols="1,2"]
|===
|Good |Bad

|`get-weather-forecast`
|`processor1`

|`publish-order-event`
|`get-data`

|`search-customer-orders`
|`myTool`
|===
