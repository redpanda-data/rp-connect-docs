== MCP specification support

MCP servers implement the open MCP protocol for tool exposure. Only the tool concept from the MCP server specification is supported. Features such as MCP resources and prompts are not yet available.

For full details, see the link:https://modelcontextprotocol.io/specification/2025-06-18/server[official MCP server specification^].
