======

Streams changes from a Microsoft SQL Server database for Change Data Capture (CDC).
Additionally, if `stream_snapshot` is set to true, then the existing data in the database is also streamed too.

include::connect:components:partial$metadata/inputs/microsoft_sql_server_cdc.adoc[]

== Permissions

To use the default Microsoft SQL Server cache, the user must have permissions to create tables and stored procedures. Refer to <<checkpoint_cache_table_name, `checkpoint_cache_table_name`>> for additional details.
