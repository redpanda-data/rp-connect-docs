== Metadata

This input adds the following metadata fields to each message:

- `kafka_key`
- `kafka_topic`
- `kafka_partition`
- `kafka_offset`
- `kafka_timestamp_unix`
- `kafka_timestamp_ms`
- `kafka_tombstone_message`
- `kafka_offset_topic`
- `kafka_offset_group`
- `kafka_offset_partition`
- `kafka_offset_commit_timestamp`
- `kafka_offset_metadata`
- `kafka_is_high_watermark`
